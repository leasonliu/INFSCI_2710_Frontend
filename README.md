Pitt INFSCI 2710 Project Frontend

## Basic intro

React and webpack based frontend API!

## Prepare Environment

Assuming you have **nodejs** v6.10 installed!

1. Install dependencies

```npm install```

2. Ready to GO!

```npm run dev```
